Process capability index，意为过程能力指数，是在`生产过程`处于统计控制状态后，用来判定过程能力是否达到顾客要求的指标。简单说来过程能力指数越高，生产可能出现的不良率就越低。

过程能力指数有两种，Cp和Cpk，这里“k”是日语“片寄り（katayori）”的首字母，意为偏倚。

同样的，过程绩效指数也有两种，Pp和Ppk。

Cp，Cpk反映的是短期内过程输出满足产品质量要求的程度，Pp，Ppk则是反映过程长期的总波动满足质量要求的程度。

由于Cpk的计算考虑了中值偏倚的问题，所以现在都用Cpk来评价产线的过程能力指数。
